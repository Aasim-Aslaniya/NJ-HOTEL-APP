# NJ Residency – Hotel Management Admin App (Figma Prototype)

## Overview
NJ Residency is a Hotel Management Admin Mobile App UI Prototype designed in Figma.
It helps hotel administrators manage daily operations such as room status, staff attendance, reservations, and guest details in one platform.
The prototype focuses on a clean layout with easy navigation, making the experience smooth for admins.
This design serves as a strong base for future Flutter development and backend integration.

---

## Project Objective

The objective of this project is to design a modern and user-friendly admin hotel management mobile UI.
The prototype connects all screens using Figma interactions, showing real app navigation flow.
It aims to reduce manual work for admins by organizing tasks like bookings, attendance, and room monitoring efficiently.
The final output is a ready-to-implement UI reference for developing the application in Flutter.

---

## Key Features
- Admin Login
- Admin Dashboard (Summary + Analytics)
- Room Management (Available / Occupied / Cleaning)
- Room Details (Guest + Booking + Payment + Housekeeping)
- Employee Attendance + Staff List + Attendance Report
- Employee Management (Add Employee, View Staff Details)
- Reservations (All / In-house / Upcoming / Past)
- Guest Details (Call / Message, History & Spend)
- Notification Icon (future module)

---

## Screens Included
1. **Admin Login Screen**
   - Email & Password login
   - Sign In button

2. **Admin Dashboard Screen**
   - Summary cards (Rooms, Employees, Revenue, Complaints, Check-ins/outs)
   - Weekly occupancy chart
   - Revenue trends chart
   - Scrollable dashboard UI

3. **Room Management Screen**
   - Room list with filters:
     - All
     - Available
     - Occupied
     - Cleaning

4. **Room Details Screen**
   - Room type, price & occupancy
   - Guest details and booking info
   - Check-in / Check-out
   - Payment status
   - Housekeeping status

5. **Employee Attendance & Staff List Screen**
   - Attendance summary (Present / Late / Absent)
   - Employee list with role & shift
   - Attendance report option
   - Add employee option

6. **Reservations Screen**
   - Tabs:
     - All
     - In-house
     - Upcoming
     - Past
   - Reservation details with status

7. **Guest Details Screen**
   - Guest profile + room number
   - Call / Message buttons
   - Guest info (Phone, Email, ID)
   - History & Spend section
